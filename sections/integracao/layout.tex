\subsection{Layout dos Componentes}

Nesta etapa, é apresentada a organização física e funcional dos blocos que compõem o sistema. O objetivo desta seção é explicitar como os módulos descritos anteriormente são interconectados para formar a arquitetura completa do circuito, evidenciando a hierarquia de dados e controle.

Como abordado, a estrutura é organizada em quatro classes principais de componentes: \textit{displays}, responsáveis pela visualização do estado; \textit{registradores}, responsáveis pelo armazenamento interno e \textit{contadores}, que implementam a lógica de transição de estados.
