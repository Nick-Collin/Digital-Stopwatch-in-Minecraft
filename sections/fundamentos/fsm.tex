\subsection{Máquina de Estados Finita}

Uma máquina de estados é um modelo matemático utilizado para descrever sistemas cujo comportamento evolui ao longo do tempo em função de entradas externas e de um estado interno previamente armazenado.

Em circuitos digitais, esse modelo é fundamental para a descrição de sistemas sequenciais, nos quais a saída não depende apenas do valor atual das entradas, mas também do histórico do sistema, representado pelo seu estado interno. Dessa forma, a memória do sistema desempenha um papel essencial em sua dinâmica.

Uma Máquina de Estados Finita (FSM - \textit{Finite State Machine}) é uma classe específica de máquina de estados na qual o conjunto de estados possíveis é finito e bem definido. Cada estado representa uma configuração discreta do sistema em um dado instante, e a evolução entre estados ocorre de maneira determinística, conforme regras de transição.

De forma intuitiva, uma FSM pode ser entendida como um sistema que:
\begin{itemize}
    \item observa uma entrada (ou ausência dela),
    \item consulta seu estado atual,
    \item e então determina o próximo estado com base em uma regra fixa.
\end{itemize}

Esse comportamento pode ser formalizado por uma quíntupla:

\[
FSM = (Q, \Sigma, \delta, q_0, F)
\]

onde:

\begin{itemize}
    \item $Q$ é o conjunto finito de estados possíveis do sistema;
    \item $\Sigma$ é o conjunto de símbolos de entrada (alfabeto de entrada);
    \item $\delta : Q \times \Sigma \rightarrow Q$ é a função de transição de estados, que define como o sistema evolui de um estado para outro;
    \item $q_0 \in Q$ é o estado inicial, que define a configuração de partida do sistema;
    \item $F \subseteq Q$ é o conjunto de estados finais ou de aceitação (em sistemas cíclicos, como contadores, este conjunto pode ser vazio ou implícito).
\end{itemize}

Em sistemas digitais sequenciais, como contadores, registradores e controladores, a FSM frequentemente opera de forma cíclica, isto é, após um conjunto de transições, o sistema retorna a estados previamente visitados, formando um ciclo contínuo de execução.
